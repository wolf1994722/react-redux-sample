# React Redux Example

* Found in [The Road to learn React](https://roadtoreact.com/)

## Features

* uses:
  * React (create-react-app)
  * Redux

## Installation

* `git clone git@github.com:rwieruch/react-redux-example.git`
* `cd react-redux-example`
* `npm install`
* `npm start`
* visit http://localhost:3000/
